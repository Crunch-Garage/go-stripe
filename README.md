# go-stripe
A project on the implementation of Stripe Api using Go for a simple payment gateway.
## Getting Started
Clone the repository:
```
$ git clone https://github.com/Crunch-Garage/go-stripe.git
```

Get into project directory:
```
$ cd go-stripe/
```

Install the project dependancies:
``` 
$ go mod tidy
```

Run project:
```
$ go run main.go
```
